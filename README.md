## 🎯 О проекте

**Shopping List Manager** — это веб-приложение, которое помогает пользователям планировать покупки, избегать забытых товаров и контролировать бюджет. Приложение поддерживает совместный доступ к спискам, что удобно для семьи или коллег.

### 🔍 Проблемы, которые решает проект:
| Проблема | Решение |
|----------|---------|
| Забывание товаров в магазине | Централизованный цифровой список с синхронизацией |
| Дублирование покупок | Совместный доступ в реальном времени |
| Неудобство бумажных заметок | Адаптивный веб-интерфейс + PWA |
| Отсутствие контроля бюджета | Автоподсчёт примерной стоимости корзины |

### 👥 Целевая аудитория:
- **Пользователи** — ведут личные списки, быстро добавляют товары
- **Семьи/Друзья** — общий доступ к одному списку, синхронизация изменений
- **Администраторы** — управление пользователями и системная аналитика

## ✨ Возможности

### Для пользователя:
- ✅ Авторизация через email или Google
- ✅ Создание, редактирование и удаление списков покупок
- ✅ Добавление товаров с категорией, количеством и примерной ценой
- ✅ Отметка купленных товаров (чекбоксы/свайпы)
- ✅ Совместный доступ: приглашение по email с ролями (редактор/просмотр)
- ✅ Автоматический подсчёт общей суммы и прогресса списка
- ✅ Экспорт завершённых списков в PDF

### Для администратора:
- 🔐 Управление пользователями (блокировка, просмотр)
- 📊 Системная статистика: активность, количество списков
- 🛡️ Модерация контента и категорий

## 🛠 Технологический стек

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **Frontend** | React 18 + Vite | Быстрая сборка, компонентная архитектура |
| **UI** | Tailwind CSS + Headless UI | Адаптивный дизайн, доступность |
| **State** | Zustand | Лёгкое управление состоянием |
| **Backend** | Node.js + Express | REST API + WebSocket для real-time |
| **Database** | PostgreSQL | Надёжное хранение реляционных данных |
| **ORM** | Prisma | Типизация, миграции, валидация схем |
| **Auth** | JWT + bcrypt | Безопасная авторизация |
| **Deploy** | Vercel + Render | Бесплатный хостинг для учебных проектов |
