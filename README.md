# 🛒 Сервис управления списком покупок (Shopping List Manager)

> Веб-приложение для управления списками покупок с возможностью добавления, отметки покупок и удаления товаров.

---

## 📋 Описание

Проект представляет собой простое веб-приложение, реализованное на **Python + Flask** с использованием **SQLite** в качестве базы данных. Пользователь может:
- Добавлять товары в список покупок.
- Отмечать товары как купленные.
- Удалять товары из списка.

---

## 🛠 Технологический стек

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **Backend** | Python + Flask | Обработка HTTP-запросов, бизнес-логика |
| **Database** | SQLite | Хранение данных в файле `shopping.db` |
| **Frontend** | HTML5, CSS3, Bootstrap 5 | Адаптивный интерфейс |
| **Template Engine** | Jinja2 | Динамическая генерация HTML |

## 🚀 Запуск проекта

1. Убедитесь, что установлен Python 3.x.
2. Установите зависимости:
   pip install -r requirements.txt
3. Запустить приложение:
   python app.py
4. Откройте браузер и перейдите по адресу http://127.0.0.1:5000.
